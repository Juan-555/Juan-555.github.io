POWER BI

The Data

The data set are the result of a survey; the participants anwers question related to fields such as Data Analyst, Data Engineering, Data Sciece and IT. Some of the fields contein information about there salary, career, favorite programing language among others.

With the data set i'm going to create a dashboard, using a column chart i'll display which is the most voted programing language, using bar chart i'll see which is the best pay job among participants; using a pie chart i'll see what the survey taker want went switching job, and i'll display how happy are the participants with there current salary and there life/ work balance.

Steps

  • First i take the data to the POWER BI Editor.
  • I start deleting several columns that i'm not going to use in the dashboard.
  • Then I split some columns by delimiters such as "/" or "(" so that the information in the columns could be better displayed in the dashboard.
  • After that I replaced the non-digits values in the column "currently yearly salary (in usd) " and made an average of the result.
