Checking at the result, i have many null fields in the columns that i'll be using; so i'll create a copy of the dataset that contains only the rows with usable values.

 #drop Null and NA
Clean = MovieData.copy()
Clean.dropna(subset=["budget"], inplace=True)
Clean.dropna(subset=["gross", "votes"], inplace=True)
Fix the decimal type values. 

Clean["votes"] = Clean["votes"].astype('int64')
